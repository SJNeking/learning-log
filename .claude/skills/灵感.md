# 顿悟捕获器

## 1. 先在对话中展开

先完整阐述灵感（≥500字），用户在终端可以看到全部内容。

## 2. 管道入库

```bash
# ✅ 用管道，内容不截断
echo "完整灵感内容..." | learnlog record "主题" --pipe --energy 5 --aha true --source "aha-capture"

# ❌ 不要这样——长文会被 shell 截断
learnlog record "主题" "灵感内容" --energy 5 --aha true
```
