# 图谱重构进度报告

## 已完成工作

### Phase 1: 基础重构 ✅

#### 1.1 数据结构定义 (`frontend/types/graph.ts`)
- 定义了完整的图谱类型系统：
  - `EnhancedGraphNode`: 增强节点类型，支持坐标、可视化状态、搜索匹配等
  - `EnhancedGraphEdge`: 增强边类型，支持关联类型、颜色、宽度等
  - `EnhancedGraphCluster`: 聚类类型，支持凸包坐标、统计信息
  - `EnhancedGraphData`: 增强图谱数据类型
  - `EnhancedGraphFilter`: 筛选条件类型
  - `EnhancedViewConfig`: 视图配置类型
- 定义了视图类型枚举：`force`（力导向图）、`timeline`（时间线）、`galaxy`（星系图）
- 定义了事件类型和常量
- 重新导入了 `AttentionGraph`、`AttentionNode`、`AttentionEdge`、`ResearchType`、`Entry` 类型

#### 1.2 工具函数库 (`frontend/lib/graph-utils.ts`)
- **颜色系统**：
  - 聚类颜色调色板（10种颜色）
  - 关联类型颜色映射（内容-蓝色、标签-绿色、时间-橙色）
  - 研究类型颜色映射
- **视觉映射**：
  - `calculateNodeSize()`: 根据能量和度数计算节点大小
  - `calculateEdgeWidth()`: 根据权重计算边宽度
- **数据转换**：
  - `transformAttentionGraph()`: 将后端数据转换为增强格式
  - 自动提取聚类信息
- **筛选逻辑**：
  - `applyFilter()`: 支持研究类型、能量等级、时间范围、标签、聚类筛选
  - `searchNodes()`: 支持标题、摘要、标签搜索
- **布局计算**：
  - `calculateTimelineLayout()`: 时间线布局算法
  - `calculateGalaxyLayout()`: 星系图布局算法（BFS分层）
  - `calculateConvexHull()`: 凸包计算（Graham scan）
- **统计计算**：
  - `calculateGraphStats()`: 计算节点数、边数、聚类数、平均度数、平均能量、密度

#### 1.3 图谱页面重构 (`frontend/app/graph/page.tsx`)
- **三视图切换**：
  - 力导向图：优化的力导向布局，支持拖拽、缩放
  - 时间线：按时间排序，交错排列，展示知识演化
  - 星系图：以最高能量节点为中心，BFS分层辐射布局
- **工具栏**：
  - 视图切换按钮（力导向图/时间线/星系图）
  - 筛选器（研究类型、能量等级、时间范围）
  - 显示控制（边显示/隐藏、标签显示/隐藏）
  - 搜索框（实时搜索，高亮匹配）
  - 刷新按钮
- **视觉增强**：
  - 节点大小根据能量和度数动态计算
  - 边的颜色根据关联类型映射（内容/标签/时间）
  - 边的样式根据权重（实线/虚线）
  - 搜索匹配节点金色高亮
  - 选中/悬停节点的关联高亮效果
  - 聚类图例和关联类型图例
- **交互增强**：
  - 点击节点显示关联面板
  - 关联节点按权重排序
  - 点击关联节点可查看详情
  - 悬停显示 tooltip
- **统计信息**：
  - 底部图例显示节点数、边数、聚类数
  - 搜索匹配数

## 重构前后对比

| 功能 | 重构前 | 重构后 |
|------|--------|--------|
| 视图模式 | 仅力导向图 | 力导向图 + 时间线 + 星系图 |
| 聚类系统 | 4个硬编码聚类 | 动态聚类（支持扩展） |
| 筛选功能 | 无 | 研究类型、能量等级、时间范围 |
| 搜索功能 | 无 | 实时搜索，高亮匹配 |
| 边的视觉 | 仅粗细变化 | 颜色（关联类型）+ 样式（权重） |
| 节点大小 | 仅度数 | 能量 + 度数 |
| 图例信息 | 简单聚类计数 | 聚类 + 统计 + 关联类型 |
| 关联面板 | 简单列表 | 按权重排序 + 关联强度标签 |

## 技术债务

2. **星系图的中心节点选择**：当前选择能量最高的节点，可以考虑让用户选择
3. **边的类型判断**：当前基于 heads 的最大值判断，可以考虑更复杂的规则
