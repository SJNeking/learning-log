- 正弦波交错排列（代替随机排列）

#### 3.4 星系图视图优化
- 支持右键点击节点设为焦点
- 选中面板"星系聚焦"按钮
- 中心节点 1.5x 放大 + 金色边框
- 4 层 BFS 展开（之前 3 层）
- 每层内随机偏移避免重叠

#### 3.5 选中节点面板增强
- 查看详情按钮
- 星系聚焦按钮
- 摘要展示区域
- 关联节点按权重排序
- 更紧凑的视觉设计

#### 3.6 键盘快捷键
| 按键 | 功能 |
|------|------|
| Escape | 取消选中/关闭面板 |
| ⌘/Ctrl + F | 聚焦搜索框 |
| 1/2/3 | 切换视图（力导向/时间线/星系） |
| R | 刷新图谱 |

#### 3.7 动画和过渡
- 力导向图：500ms cubicOut
- 时间线：800ms cubicOut
- 星系图：600ms cubicOut
- 选中节点 1.3x 放大 + 金色边框
- 搜索匹配节点金色高亮
